#  📌 jikgong(직공)
> 건설 노동자와 기업들의 일자리 매칭을 시켜주는 중개플랫폼 입니다.


<br><br>

## 🛠 stacks
### Backend
- Java 17
- Spring Boot 3.0, JPA(Hibernate)
- Spring Security
- Spring Batch
- Spring Event
- Spring Actuator
- QueryDsl
- MySQL 8.0.28
- Redis

### DevOps
- Github Action
- Docker
- AWS (EC2, RDS, S3, Cloud Watch)
- Nginx
- Prometheus, Grafana, Slack

### 협업 툴
- Git, Github
- Figma
- Swagger
- Notion
